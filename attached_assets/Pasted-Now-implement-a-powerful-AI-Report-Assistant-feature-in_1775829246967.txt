Goal:
Help admin turn messy raw notes into clean, structured, and usable reports.

This should become one of the most useful tools in the system.

==================================================
MAIN FUNCTION
==================================================

User can paste raw text (messy notes), and AI will transform it into structured output.

==================================================
INPUT
==================================================

- textarea input
- user pastes raw notes (meeting notes, updates, ideas, etc.)

==================================================
OUTPUT MODES
==================================================

Create multiple selectable output modes:

1. Notulensi Mode
Output structure:
- Judul
- Tanggal
- Peserta
- Ringkasan
- Keputusan
- Tindak Lanjut

2. Internal Report Mode
Output structure:
- Fokus Update
- Perkembangan Utama

- Insight
- Next Step

3. Investor Summary Mode
Output structure:
- Highlight Utama
- Progress Penting
- Dampak Strategis
- Arah Selanjutnya

4. Short Summary Mode
Output:
- concise bullet summary
